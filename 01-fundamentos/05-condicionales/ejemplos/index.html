Ejercicios de Condicionales

Aquí practicarás lo aprendido. Abre la consola (F12) para ver los resultados.

Ejercicio 1: Positivo, negativo o cero

Completa la función para que devuelva "POSITIVO", "NEGATIVO" o "CERO"

Ejercicio 2: ¿Puede votar?

Edad mínima para votar: 16 años

Ejercicio 3: Calificación con letra

90+ = A, 80-89 = B, 70-79 = C, 60-69 = D, menor = F

Ejercicio 4: Semáforo

"verde" = "Avanzar", "amarillo" = "Precaución", "rojo" = "Detenerse"

Ejercicio 5: Año bisiesto

Un año es bisiesto si es divisible entre 4, excepto si es divisible entre 100 pero no entre 400
